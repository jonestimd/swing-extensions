Custom Components

Date field with calendar popup

  • Configurable date format
  • Localized (month/day names and first day of week)
  • Support for use as a table cell editor

Bean list combo box

  • Renders list items using a java.text.Format
  • Select an item by typing initial characters of the display value
  • Display feedback for a required value
  • Support for use as a table cell editor

Validated panel

  • Display validation messages for fields
  • Display validation messages for table cells

Table Extensions

  • Bean list table model
    • Table data as a list of Java beans
    • Table columns correspond to bean properties
    • Define column adapters to read/write bean properties
    • Support for asynchronous data providers
  • Input validation
  • Highlight unsaved changes
  • Support for search/filter
    • Complex search
    • Syntax highlighting
    • Error feedback
  • Mixed row table
    • Header/detail table
    • Section table
